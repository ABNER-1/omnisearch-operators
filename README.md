## QUICK START
以 face embedding encoder 为例, 本节旨在3分钟之内搭建一个最简单的 encoder 服务.
```shell
# 切换到工作目录
cd face-encoder
# 运行 quick_start 脚本, 包括
# 1. 准备模型
# 2. docker 环境检查
# 3. 构建 docker 镜像
# 4. 启动 docker 容器
./quick_start.sh
```
